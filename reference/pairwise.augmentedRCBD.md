# Perform Pairwise t Tests of Adjusted Means from `augmentedRCBD` Output

`pairwise.augmentedRCBD` performs pairwise t tests of adjusted means
from an object of class `augmentedRCBD`.

## Usage

``` r
pairwise.augmentedRCBD(
  aug,
  cl = NULL,
  alpha = NULL,
  p.adjust = c("none", "tukey", "sidak")
)
```

## Arguments

- aug:

  An object of class `augmentedRCBD`.

- cl:

  A cluster object created by
  [`makeCluster`](https://rdrr.io/r/parallel/makeCluster.html) for
  parallel evaluations.

- alpha:

  Type I error probability (Significance level).

- p.adjust:

  The p value adjustment method. Either `"none"`, `"tukey"` or
  `"sidak"`.

## Value

A data frame of pairwise comparisons of treatments.

## Details

The default pairwise comparison in
[`augmentedRCBD`](https://aravind-j.github.io/augmentedRCBD/reference/augmentedRCBD.html)
employs
[`pairs.emmGrid`](https://rvlenth.github.io/emmeans/reference/contrast.html)
function from
[`emmeans`](https://rvlenth.github.io/emmeans/reference/emmeans.html)
which is very slow for large number of comparisons. This function
attempts to do the same faster with parallel computing with the package
[`parallel-package`](https://rdrr.io/r/parallel/parallel-package.html).

## See also

[`makeCluster`](https://rdrr.io/r/parallel/makeCluster.html)
